# FakeTwitter Clone

A static Twitter Timeline clone built with HTML, CSS, and JavaScript.

## Overview

This project recreates a Twitter-like home timeline experience, including:
- left navigation sidebar
- tweet composer with character count
- interactive tweet actions
- trending topics panel
- featured video card using `assets/images/orlando.mp4`
- dark mode toggle and tweet popup features

## Files

- `html/index.html` — main page markup
- `css/style.css` — design and responsive layout
- `js/app.js` — tweet rendering and interaction logic
- `js/dark-mode.js` — manual dark mode toggle feature
- `js/trending.js` — trending topics UI and refresh behavior
- `js/tweet-popup.js` — tweet composer popup interaction
- `assets/images/orlando.mp4` — featured video asset

## How to Use

1. Open `html/index.html` directly in a browser.
2. Use the tweet composer to enter sample text and post a tweet.
3. Click trending topics to populate the tweet box.
4. Toggle dark mode and open the tweet popup to test additional features.

## Notes

- This is a static front-end project only, with no server or backend.
- The app is optimized for responsive display on desktop and mobile screen sizes.
- All behavior is handled in plain JavaScript without external frameworks.
